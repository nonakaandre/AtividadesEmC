# Atividades

Repositório com atividades, exercícios e práticas desenvolvidas ao longo do curso de Desenvolvimento de Software Multiplataforma (DSM).

## 📌 Objetivo

Organizar e registrar as atividades realizadas durante o curso, com foco no desenvolvimento da lógica de programação e fundamentos da computação.

## 🧠 Contexto

A maior parte das atividades deste repositório foi desenvolvida utilizando a linguagem C, com foco em:

- Lógica de programação
- Estruturas de controle

- Organização de código

## 🛠️ Tecnologias utilizadas

- C (principal)
- Outras tecnologias podem aparecer pontualmente conforme a atividade

## 📂 Estrutura do repositório

Os arquivos estão organizados conforme atividades ou temas abordados.
